## Challenge: Database Upload

### Technologies used
<ul>
  <li>Node.js</li>
  <li>TypeScript</li>
  <li>Prettier</li>
  <li>ESLINT</li>
  <li>Jest</li>
  <li>EditorConfig</li>
  <li>Postgres</li>
</ul>

You will need to create a database for this project, configs on ormconfig.json file.

### `yarn`
To install all necessary packages

### `yarn dev::server`
To run a developmet server

### `yarn test`
To execute the app tests.
